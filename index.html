    • CCSP (ISC)2 Certified Cloud Security Professional Official Study Guide (2019)
      • - Ben Malisow, Sybex.
- ISBN: 9781119277415

Wiley Education offered me the opportunity to review the "ISC2 CCSP Certified Cloud 
Security Professional Official Study Guide" by Ben Malisow 
(ISBN-13: 978-1119603375, ISBN-10: 1119603374, 2th edition - 2019) as a possible 
candidate reference to be used by our internal students and as a guide to support 
corporate cloud and cybersecurity learning tracks.

As a cyber security professional and trainer, I found the book to be very helpful to 
obtain an overview of the body of content related to the ISC2 CCSP in a short time. 
In between work, I was able to finish it in about 14 days.

If you are new to cloud security essentials, the book provides a clear explanation 
of the most important topics, terminology and concepts that are vital for the exam. 

As an exam guide should be, it finds the right balance between required explanations, 
to the point topic summaries and lists of references – that you, as an exam candidate, 
know you should google on. It also pays attention to references to EU law, such as the GDPR.

Added value: if you are new to CBTs, the book provides access to online example questions 
and flash cards, all easy to use through your web browser.

Although Ben explicitly states that “you cannot expect to pass the exam using this book as your sole source” 
- if you already have a cyber security background and experience with cloud fundamentals – you have a fair chance ??
